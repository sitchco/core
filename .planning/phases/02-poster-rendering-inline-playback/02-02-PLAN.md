<task type="auto">
  <name>Task 2: Create view.js with click-to-play SDK loading, dimension locking, and player creation</name>
  <files>modules/VideoBlock/blocks/video/view.js</files>
  <action>
    Create `view.js` as a vanilla JavaScript file (no imports -- all dependencies come from `window.sitchco`). The file handles click-to-play behavior for both YouTube and Vimeo providers.

    **Initialization:**
    Use `sitchco.register()` to register a callback that runs on DOMContentLoaded:
    ```javascript
    sitchco.register(function initVideoBlocks() {
        document.querySelectorAll('.sitchco-video[data-url]').forEach(initVideoBlock);
    });
    ```

    **initVideoBlock(wrapper):**
    Read data attributes from the wrapper element:
    - `dataset.url`, `dataset.provider`, `dataset.videoId`, `dataset.displayMode`, `dataset.clickBehavior`

    Skip initialization for non-inline display modes (modal/modal-only will be handled in Phase 3).

    Determine click target:
    - If `clickBehavior === 'poster'`: click target = wrapper element itself
    - If `clickBehavior === 'icon'`: click target = `.sitchco-video__play-button` inside wrapper

    Add click handler AND keydown handler (for Enter/Space on the wrapper when `role="button"`) to the click target. The handler calls `handlePlay(wrapper)`. Use `{ once: true }` option on the event listener so it only fires once.

    For keyboard: when wrapper has `role="button"`, listen for `keydown` on wrapper and call handlePlay if key is Enter or Space (prevent default for Space to avoid page scroll).

    **handlePlay(wrapper):**
    1. INLN-02: Lock dimensions BEFORE any DOM changes:
       ```javascript
       wrapper.style.width = wrapper.offsetWidth + 'px';
       wrapper.style.height = wrapper.offsetHeight + 'px';
       ```

    2. INLN-03: Hide poster and play button:
       ```javascript
       wrapper.classList.add('sitchco-video--playing');
       ```

    3. Remove interactive attributes (they are no longer needed after activation):
       ```javascript
       wrapper.removeAttribute('role');
       wrapper.removeAttribute('tabindex');
       wrapper.removeAttribute('aria-label');
       ```

    4. Create player container:
       ```javascript
       const playerContainer = document.createElement('div');
       playerContainer.className = 'sitchco-video__player';
       wrapper.appendChild(playerContainer);
       ```

    5. Extract start time from URL and create player based on provider:
       ```javascript
       const provider = wrapper.dataset.provider;
       const url = wrapper.dataset.url;
       const videoId = wrapper.dataset.videoId;
       if (provider === 'youtube') {
           createYouTubePlayer(playerContainer, videoId, extractYouTubeStartTime(url));
       } else if (provider === 'vimeo') {
           createVimeoPlayer(playerContainer, videoId, extractVimeoStartTime(url));
       }
       ```

    **YouTube IFrame API Singleton Loader:**
    ```javascript
    let ytAPIPromise = null;

    function loadYouTubeAPI() {
        if (ytAPIPromise) return ytAPIPromise;
        if (window.YT && window.YT.Player) {
            ytAPIPromise = Promise.resolve(window.YT);
            return ytAPIPromise;
        }
        ytAPIPromise = new Promise((resolve) => {
            const prev = window.onYouTubeIframeAPIReady;
            window.onYouTubeIframeAPIReady = () => {
                if (prev) prev();
                resolve(window.YT);
            };
            sitchco.loadScript('youtube-iframe-api', 'https://www.youtube.com/iframe_api');
        });
        return ytAPIPromise;
    }
    ```

    **createYouTubePlayer(container, videoId, startTime):**
    ```javascript
    async function createYouTubePlayer(container, videoId, startTime) {
        const YT = await loadYouTubeAPI();
        new YT.Player(container, {
            videoId: videoId,
            host: 'https://www.youtube-nocookie.com',  // PRIV-02
            playerVars: {
                autoplay: 1,       // INLN-05
                playsinline: 1,
                enablejsapi: 1,
                origin: window.location.origin,
                start: startTime,  // INLN-06
                rel: 0,
            },
            events: {
                onReady: function(event) {
                    event.target.playVideo();  // INLN-05 belt-and-suspenders
                },
            },
        });
    }
    ```

    **Vimeo SDK Loader + Player:**
    ```javascript
    function loadVimeoSDK() {
        return sitchco.loadScript('vimeo-player', 'https://player.vimeo.com/api/player.js');
    }

    async function createVimeoPlayer(container, videoId, startTime) {
        await loadVimeoSDK();
        const player = new Vimeo.Player(container, {
            id: parseInt(videoId, 10),
            autoplay: true,   // INLN-05
            dnt: true,         // PRIV-03
        });
        if (startTime > 0) {
            player.ready().then(function() {
                player.setCurrentTime(startTime);
            });
        }
    }
    ```

    **Start Time Extraction:**
    Implement `extractYouTubeStartTime(url)` and `extractVimeoStartTime(url)` following the patterns from RESEARCH.md Code Examples section.

    YouTube: Parse `?t=`, `?start=`, or `&t=` parameter. Handle formats: plain seconds ("90"), seconds suffix ("90s"), minutes+seconds ("1m30s"), hours+minutes+seconds ("1h2m30s"). Return integer seconds, default 0.

    Vimeo: Parse `#t=` hash fragment. Handle seconds format ("90s" or "90"). Return integer seconds, default 0.

    **Video ID Extraction (not needed in view.js):** The video ID is already extracted server-side in render.php and provided as `data-video-id`. view.js reads it from `wrapper.dataset.videoId`. Do NOT duplicate ID extraction in JavaScript.

    **Error handling:** Wrap SDK loading in try/catch. On failure, log to console.error. Do not throw -- the poster is already hidden, so a failure is visible but should not crash the page.

    **Code style:** Use vanilla JavaScript (no TypeScript, no JSX). Function declarations (not arrow functions at top level) per project conventions. Use `function` keyword for named functions. Arrow functions are fine for short callbacks.
  </action>
